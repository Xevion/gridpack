# Gridpack

A small playground for image layout algorithms.

Gridpack renders the same set of images through several layout engines — justified,
masonry, square grid, linear partition, Knuth-Plass, bin packing, and treemap — and
lets you tweak their parameters live to see how each one packs the grid. It's built
with SvelteKit, Panda CSS, and Ark UI, and deploys as a static site to GitHub Pages.
